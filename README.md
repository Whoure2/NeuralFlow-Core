# NeuralFlow-Core

![GitHub License](https://img.shields.io/github/license/Whoure2/NeuralFlow-Core) ![GitHub issues](https://img.shields.io/github/issues/Whoure2/NeuralFlow-Core)

High-performance library for custom neural network architectures in Python and C++.

## Features
- Customizable layers and activation functions.
- GPU acceleration with CUDA support.
- Seamless integration with existing ML frameworks.

## Installation
```bash
pip install neuralflow-core
```
